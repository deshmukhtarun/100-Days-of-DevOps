# 100 Days of DevOps

This repository documents my **100 Days of DevOps learning journey** through hands-on labs and real-world scenarios.

The goal of this challenge is to build strong practical knowledge in:

* Linux administration
* Networking fundamentals
* Docker
* Kubernetes
* CI/CD pipelines
* Infrastructure as Code
* Cloud platforms

Each day includes:

* The task description
* Commands used to solve the problem
* Notes explaining the concepts learned

This repository acts as both a **learning journal** and a **DevOps portfolio**.
