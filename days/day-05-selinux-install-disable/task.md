# Day 05 – Install and Disable SELinux

## Task

Following a security audit, the **xFusionCorp Industries security team** decided to enhance system security using **SELinux**.

For testing purposes, SELinux must be installed and temporarily disabled on **App Server 2** in the **Stratos Datacenter**.

## Requirements

1. Install the required **SELinux packages**.
2. **Permanently disable SELinux**.
3. **Do not reboot the server**, as a maintenance reboot is already scheduled.
4. Ignore the current SELinux status from the command line; after the next reboot the status must be **disabled**.
