# Day 02 – Create Temporary User with Expiry Date

## Task

As part of the temporary assignment to the **Nautilus project**, a developer named **mariyam** requires system access for a limited duration.

To manage this requirement properly, a temporary user account must be created with a defined expiry date.

## Requirements

Create a user named **mariyam** on **App Server 3** in the **Stratos Datacenter**.

Conditions:

* Username must be **lowercase**
* The account must have an **expiry date**
* Expiry date must be **2027-03-28**

This ensures the account will automatically become inactive after the specified date.
