# Day 11 – Tomcat Deployment

## Task

The Nautilus development team has built a Java-based application and wants to deploy it using **Tomcat** on App Server 1.

## Requirements

1. Install Tomcat on App Server 1
2. Configure Tomcat to run on port **8083**
3. A `ROOT.war` file is available on Jump Host at `/tmp`
4. Deploy the application so it is accessible at:

```bash
http://stapp01:8083
```

The application should work directly on the base URL.
