# Day 04 – Grant Execute Permission to Script

## Task

The xFusionCorp Industries sysadmin team has developed a new backup automation script named **xfusioncorp.sh**.

The script has been distributed to all required servers but currently lacks executable permissions on **App Server 2** within the **Stratos Datacenter**.

## Requirements

Grant executable permissions to the script located at:

/tmp/xfusioncorp.sh

Ensure that **all users on the system can execute the script**.
