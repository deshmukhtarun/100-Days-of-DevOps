# Day 03 – Disable Direct Root SSH Login

## Task

Following recent security audits, the **xFusionCorp Industries security team** introduced new security protocols to strengthen server access control.

One of the required changes is disabling **direct root SSH login**.

## Requirements

Disable direct **SSH root login** on **all app servers** within the **Stratos Datacenter**.

This change ensures administrators must log in using regular user accounts and then elevate privileges when required.
