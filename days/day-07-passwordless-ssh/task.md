# Day 07 – Password-less SSH Setup

## Task

The system admins team at **xFusionCorp Industries** has configured scripts on the **jump host** that execute tasks on all app servers in the Stratos Datacenter.

To ensure smooth execution, password-less SSH access must be configured.

## Requirements

Set up password-less SSH authentication:

* From user **thor** on the jump host
* To all app servers
* Using their respective sudo users (e.g., **tony** for App Server 1)

This allows automated scripts to run without requiring manual password input.
