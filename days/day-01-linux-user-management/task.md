# Day 01 – Create Non-Interactive User

## Task

To accommodate the backup agent tool's specifications, the system admin team at **xFusionCorp Industries** requires the creation of a user with a non-interactive shell.

### Requirements

Create a user named **rose** with a **non-interactive shell** on **App Server 1**.

The user should exist on the system but must **not be able to log in interactively**.

This type of configuration is commonly used for **service accounts used by automation tools or system services**.
